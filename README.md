# Finding Missing Person Using AI

Finding Missing Person Using AI is an AI-based web application designed to help locate missing individuals using facial recognition.  
The system uses MediaPipe Face Mesh to extract facial features from images and compares them with stored missing person records.

### Features
- AI-based face recognition  
- Missing person registration  
- Automatic face matching  
- Web and mobile-friendly interface  
- SQLite database for storage  

### Tech Stack
Python, Streamlit, MediaPipe, OpenCV, SQLite

### How to Run
```bash
pip install -r requirements.txt
streamlit run Home.py
Use Case

Helps police, NGOs, and authorities to identify missing persons faster using AI.
